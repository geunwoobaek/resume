### 메이크스타(MakeStar) → [상세보기](detail/메이크스타_상세.md)
> 백엔드 엔지니어 | 2024.05 ~ 현재

#### OMS(Order Management System) 백엔드 개발

K-POP 커머스 플랫폼의 주문 관리 시스템 백엔드 설계 및 개발 담당

기술 스택 : Java 21, Kotlin, Spring Boot 3.x, JPA/QueryDSL, MyBatis, MySQL(Aurora), Redis, GKE, Google Cloud Tasks/Scheduler, Flyway, Sentry

- 주문 생성/조회/수정/상태 관리 등 핵심 주문 처리 API 개발
- QueryDSL 기반 동적 쿼리를 활용한 복합 조건 주문 검색 기능 구현
- Google Cloud Tasks를 활용한 비동기 배치 처리 시스템 구축 (대량 주문 할당, 리셋, WMS 연동 등)
- 재고 할당 시스템 개발 (자동 할당 / 수동 할당 / 시뮬레이션 모드 지원)
- Pessimistic Locking + Lock Ordering 기반 재고 할당 동시성 제어 설계 (Deadlock 방지, Atomic Update, JDBC Batch)
- OpenFeign + TransactionalEventListener(BEFORE_COMMIT)를 활용한 커머스 백엔드 양방향 연동 (주문 상태, 할당 결과, SKU 치수, 운송사 정보 콜백)
- Flyway를 이용한 DB 마이그레이션 체계 구축

#### 배송비 산정 시스템 개발

10개 이상의 국제 운송사를 지원하는 동적 배송비 산정 로직 설계 및 개발

- 목적지 국가, 무게, 박스 타입, 운송사별 다단계 배송비 계산 로직 구현
- 3D Bin Packing 알고리즘을 활용한 박스 최적화 패킹 시스템 개발
- 운송사별(한진, FedEx, UPS, EMS, SF Express 등) 배송 정책 관리 기능 개발

#### TMS(Transportation Management System) 개발

주문 배송 추적 및 물류 연동을 위한 운송 관리 시스템 개발

기술 스택 : Kotlin, Spring Boot 3.x, GKE, Google Cloud Scheduler

- SF Express, GoodsFlow 등 외부 물류사 API 연동을 통한 실시간 배송 추적 시스템 구축
- Cloud Scheduler 기반 주기적 배송 상태 동기화 파이프라인 구현
- 외부 물류사 콜백 수신 및 주문 상태 자동 업데이트 처리

#### Java 21 마이그레이션 및 Kotlin 전환

레거시 시스템의 현대화 작업 수행

- Java 15 → Java 21 LTS 업그레이드 (javax → jakarta EE 마이그레이션 포함)
- Java 코드의 Kotlin 전환 작업 진행
- Spring Boot 3.x, Spring Security 6.x 업그레이드 대응

#### 인프라 및 CI/CD

- GKE 기반 멀티 환경(dev/stage/prod) 배포 파이프라인 구축
- GitHub Actions를 활용한 CI/CD 자동화 (lint, test, build, deploy)
- Sentry 기반 에러 모니터링 및 Google Cloud Profiler를 활용한 APM 구축
- Docker 컨테이너화 및 환경별 프로파일 관리
